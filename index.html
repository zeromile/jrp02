Job Readiness Program, Phase 1 Syllabus

West Hills College and Geekwise Academy

Course Description

Phase 1 of the Job Readiness Program is an intensive-instruction class, designed to further develop existing web development skills already learned in the West Hills College CTE program. Because students will have this base knowledge there won’t be a need for Geekwise to cover these topics in detail. However, the initial part of Phase 1 will require instructor and students to review HTML, CSS and Javascript to assess students’ skill and aptitude in these areas. Post review, students will delve further into Javascript and MEAN Stack. Some students may still need to be brought up to speed on the basics and we will do our best to accommodate.

Course Technology

Students will be required to bring their own laptop computer. The computer must be in good working condition able to run the latest browser and editing software. We will primarily use the Chrome browser, Git version control and the Atom text editor. Student must have “Administrator” privileges in order to install software.

Computer Usage

In order to stay focused, please refrain from activities that are distracting to yourself and others around you. Everyone should have an equal opportunity to learn in a distraction-free environment.

Classroom Behavior Expectations

  • Please respect our building. You are expected to clean up after yourself.
  • Cell phones should be on silent mode during the class.
  • Help each other. This class is about making you better so you can make your community better. We want to foster an environment of cooperation and assistance.
